# What is Network Security?

Network Security is the practice of protecting computer networks from unauthorized access, attacks, misuse, modification, or destruction.

It ensures that only authorized users can access network resources while maintaining the confidentiality, integrity, and availability (CIA Triad) of data.

---

# Why is Network Security Important?

Without proper network security:

- Hackers can steal sensitive data.
- Malware can spread across systems.
- Services can become unavailable.
- Organizations may suffer financial and reputational losses.

Network Security helps:

- Protect sensitive information.
- Prevent cyber attacks.
- Maintain business continuity.
- Ensure secure communication.

---

# Goals of Network Security

- Confidentiality
- Integrity
- Availability
- Authentication
- Authorization
- Accountability

# Types of Network Security

### 1. Access Control

Restricts who can access network resources.

Example:
Only employees can access the company network.

---

### 2. Firewall

A firewall monitors and filters incoming and outgoing network traffic based on predefined security rules.

Purpose:

- Block unauthorized access
- Allow trusted traffic

---

### 3. Antivirus & Anti-malware

Detects, blocks, and removes malicious software.

---

### 4. Virtual Private Network (VPN)

Creates an encrypted connection over the Internet.

Benefits:

- Secure remote access
- Data encryption
- Privacy

---

### 5. Intrusion Detection System (IDS)

Monitors network traffic and detects suspicious activities.

---

### 6. Intrusion Prevention System (IPS)

Detects attacks and automatically blocks malicious traffic.

---

# Common Network Attacks

## 1. Denial of Service (DoS)

An attacker floods a server with requests, making it unavailable.

---

## 2. Distributed Denial of Service (DDoS)

Multiple infected systems attack one target simultaneously.

---

## 3. Man-in-the-Middle (MITM)

An attacker secretly intercepts communication between two parties.

Example:

Attacker intercepts data between your laptop and a public Wi-Fi.

---

## 4. Packet Sniffing

Capturing network packets to steal sensitive information.

---

## 5. Spoofing

Pretending to be another trusted device or user.

Examples:

- IP Spoofing
- MAC Spoofing
- Email Spoofing

---

## 6. DNS Attacks

Manipulating DNS responses to redirect users to fake websites.

# Best Practices

- Use strong passwords.
- Enable Multi-Factor Authentication (MFA).
- Keep software updated.
- Use firewalls.
- Encrypt sensitive data.
- Monitor network traffic.
- Disable unused services.
- Use VPN for remote access.

# Real-Life Example

A company allows employees to work from home.

Employees connect through a VPN.

The firewall filters malicious traffic.

IDS monitors suspicious activities.

Antivirus protects endpoints.

Together, these technologies keep the company network secure.
