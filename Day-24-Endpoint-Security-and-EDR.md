# 📖 Introduction

Endpoints are devices connected to a network.

Examples:

- Laptops
- Desktops
- Servers
- Smartphones
- Tablets

Attackers often target endpoints because they are the entry point into an organization's network.

---

# What is Endpoint Security?

Endpoint Security protects devices from cyber threats.

Objectives:

- Prevent Malware
- Detect Threats
- Respond to Attacks
- Protect Sensitive Data

---

# Endpoint Protection Platform (EPP)

Traditional endpoint security.

Features:

- Antivirus
- Firewall
- Device Control
- Malware Protection

---

# Antivirus

Antivirus detects known malware using signatures.

Advantages:

- Fast Detection
- Easy Deployment

Limitation:

Cannot always detect new threats.

---

# Endpoint Detection & Response (EDR)

EDR continuously monitors endpoint activities.

Capabilities:

- Threat Detection
- Behavioral Analysis
- Incident Investigation
- Automated Response

Unlike traditional antivirus, EDR detects suspicious behavior even if malware is unknown.

---

# Extended Detection & Response (XDR)

XDR expands protection beyond endpoints.

Monitors:

- Endpoints
- Email
- Cloud
- Network
- Identity

XDR provides a complete view of attacks across the organization.

---

# Behavioral Analysis

Instead of relying only on malware signatures, EDR monitors abnormal behavior.

Examples:

- Unusual PowerShell execution
- Mass file encryption
- Privilege escalation
- Suspicious network connections

---

# Endpoint Hardening

Methods include:

- OS Updates
- Patch Management
- Application Whitelisting
- Disk Encryption
- Strong Passwords
- MFA

---

# Common Endpoint Threats

- Malware
- Ransomware
- Phishing
- Insider Threats
- Credential Theft
- USB Attacks

---

# Popular Endpoint Security Solutions

- Microsoft Defender for Endpoint
- CrowdStrike Falcon
- SentinelOne
- Sophos Intercept X
- Trend Micro Apex One

---

# Endpoint Security Best Practices

- Keep systems updated.
- Enable BitLocker/FileVault.
- Install EDR.
- Restrict USB devices.
- Enable MFA.
- Monitor endpoint logs.

---

# 🌍 Real-Life Example

An employee accidentally downloads ransomware.

Microsoft Defender for Endpoint detects unusual file encryption.

The infected device is automatically isolated.

The SOC team investigates and removes the malware before it spreads.

---

# 📌 Key Takeaways

- Endpoints are common attack targets.
- Antivirus detects known threats.
- EDR detects suspicious behavior.
- XDR extends visibility across multiple security layers.
- Endpoint hardening reduces attack risk.

---

# ❓ Interview Questions

### 1. What is Endpoint Security?

Protecting end-user devices from cyber threats.

---

### 2. Difference between Antivirus and EDR?

Antivirus mainly detects known malware using signatures, while EDR continuously monitors endpoint behavior and detects suspicious activities.

---

### 3. What is XDR?

Extended Detection and Response integrates endpoint, network, cloud, identity, and email security.

---

### 4. Name three Endpoint Security solutions.

- CrowdStrike Falcon
- Microsoft Defender for Endpoint
- SentinelOne

---

### 5. Why is Endpoint Security important?

Because endpoints are often the first target for cyber attackers.
