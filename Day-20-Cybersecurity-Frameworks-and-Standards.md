# 📖 Introduction

Modern organizations require structured security practices to protect their systems, data, and users.

Instead of creating their own security policies from scratch, they follow internationally recognized cybersecurity frameworks.

These frameworks provide guidelines for:

- Risk Management
- Security Controls
- Incident Response
- Compliance
- Continuous Improvement

They help organizations build strong security programs.

---

# 🛡️ What is a Cybersecurity Framework?

A Cybersecurity Framework is a structured set of best practices, guidelines, and standards that help organizations manage cybersecurity risks.

Benefits include:

- Improved Security
- Better Risk Management
- Regulatory Compliance
- Consistent Security Policies
- Faster Incident Response

---

# NIST Cybersecurity Framework (CSF)

The National Institute of Standards and Technology (NIST) developed one of the world's most widely adopted cybersecurity frameworks.

The NIST CSF consists of six core functions.

```
Govern

↓

Identify

↓

Protect

↓

Detect

↓

Respond

↓

Recover
```

---

# 1. Govern

Establish organizational cybersecurity governance.

Activities include:

- Security Policies
- Risk Management
- Roles and Responsibilities
- Compliance

---

# 2. Identify

Understand organizational assets and risks.

Examples:

- Hardware
- Software
- Employees
- Cloud Resources
- Business Processes

---

# 3. Protect

Implement safeguards to reduce cybersecurity risks.

Examples:

- Access Control
- Encryption
- Firewalls
- Security Awareness Training
- MFA

---

# 4. Detect

Identify cybersecurity incidents quickly.

Examples:

- SIEM
- IDS
- Threat Monitoring
- Log Analysis

---

# 5. Respond

Take action after detecting an incident.

Activities include:

- Incident Response
- Containment
- Investigation
- Communication

---

# 6. Recover

Restore normal operations.

Examples:

- Restore Backups
- System Recovery
- Lessons Learned
- Business Continuity

---

# ISO/IEC 27001

ISO/IEC 27001 is the international standard for Information Security Management Systems (ISMS).

It helps organizations:

- Protect Information
- Manage Security Risks
- Improve Security Processes
- Meet Compliance Requirements

---

# Information Security Management System (ISMS)

An ISMS includes:

- Security Policies

- Incident Response
- Continuous Improvement

# CIS Controls

The Center for Internet Security (CIS) provides practical security controls.

Examples include:

- Inventory of Assets
- Secure Configuration
- Vulnerability Management
- Access Control
- Continuous Monitoring
- Security Awareness

CIS Controls are widely used because they are practical and easy to implement.

---

# MITRE ATT&CK Framework

MITRE ATT&CK is a knowledge base of attacker behaviors.

It helps organizations understand:

- How attackers operate
- Attack techniques
- Detection strategies
- Defensive measures

Examples of ATT&CK Tactics:

- Initial Access
- Execution
- Persistence
- Privilege Escalation
- Defense Evasion
- Credential Access
- Discovery
- Lateral Movement
- Exfiltration
- Impact

---

# Zero Trust Security

Traditional security assumes users inside the network can be trusted.

Zero Trust follows a different principle:

> **Never Trust, Always Verify**

Every request is verified regardless of location.

Zero Trust principles include:

- Verify Identity
- Least Privilege Access
- Continuous Monitoring
- Device Verification

---

# Defense in Depth

Defense in Depth uses multiple layers of security instead of relying on a single protection mechanism.

```
User

↓

MFA

↓

Firewall

↓

IDS/IPS

↓

Endpoint Protection

↓

Encryption

↓

Application Security

↓

Data Security
```

If one layer fails, other layers continue protecting the organization.

---

# Risk Management

```
↓

Analyze Risk

↓

Evaluate Risk

↓

Treat Risk

↓

Monitor Risk
```

# 🌍 Real-Life Example

A multinational company adopts the NIST Cybersecurity Framework.

It:

- Identifies critical assets.
- Protects systems using firewalls and MFA.
- Detects attacks using SIEM.
- Responds through an Incident Response Team.
- Restores systems using secure backups.

The company also follows ISO 27001 and implements Zero Trust Architecture for stronger security.
