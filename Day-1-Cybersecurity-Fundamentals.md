# Day 1 - Cybersecurity Fundamentals

## What is Cybersecurity?

Cybersecurity is the practice of protecting systems, networks, applications and data from cyber attacks.

---

## What Does Cybersecurity Protect?

### Devices
- Laptop
- Phone
- Servers

### Networks
- Wi-Fi
- Internet connections

### Applications
- Instagram
- Banking Apps
- Websites

### Data
- Passwords
- Photos
- Bank Information

---

## CIA Triad

### Confidentiality
Only authorized users should access data.

Example:
Password-protected Gmail account.

### Integrity
Data should not be modified without permission.

Example:
Bank balance should not change unexpectedly.

### Availability
Systems should be accessible when needed.

Example:
Google should work whenever users open it.

---

## Threat

Anything that can cause harm.

Examples:
- Hackers
- Malware
- Ransomware

---

## Vulnerability

A weakness in a system.

Examples:
- Weak passwords
- Outdated software
- No antivirus

---

## Risk

The chance that a threat can exploit a vulnerability.

Example:
Weak password + Hacker = High Risk

---

## Authentication

Verifying identity.

Examples:
- Password
- OTP
- Fingerprint

---

## Authorization

Determines what a user can access after login.

Example:
Admin can delete files.
Normal user cannot.

---

## Malware

Malicious software designed to harm systems.

### Types

#### Virus
Requires user action.

#### Worm
Spreads automatically.

#### Trojan
Looks safe but contains malicious code.

#### Ransomware
Encrypts files and demands money.

---

## Social Engineering

Manipulating humans to reveal information.

### Phishing

Fake:
- Emails
- SMS
- Websites

Used to steal:
- Passwords
- OTPs
- Banking details
